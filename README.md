# Nexus Laravel Vendor Adapter

Laravel adapter package for Nexus Vendor integration.

## Package

- Composer: `azaharizaman/nexus-laravel-vendor-adapter`
- Namespace: `Nexus\Adapter\Laravel\Vendor\`

## Installation

```bash
composer require azaharizaman/nexus-laravel-vendor-adapter
```

## Purpose

This package provides Laravel-specific vendor adapters, including Eloquent-backed implementations for vendor domain ports.
